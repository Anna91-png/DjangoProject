Является учебным проектом по Python-разработке. Проект содержит код для разработки простого web-приложения интернет-магазина.

Проект разрабатывается с использованием фреймворка Django

Установка:
Клонируйте репозиторий проекта:
git clone https://github.com/Anna91-png/DjangoProject
Установите зависимости внутри каталога проекта для создания виртуального окружения и установки зависимостей:
poetry install
